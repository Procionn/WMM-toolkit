# WMM-toolkit

| [English](README.md) | [Русский](README.ru.md) |
| -------------------- | ----------------------- |

This repository is a **starter template and guide** for developing extensions for [Wirus Mod Manager (WMM)](https://github.com/Procionn/WMM).
You can clone it and use it as a foundation for your own extension.

Here you will find:
*   [Documentation](doc/en/main.md) on extension development.
*   Prepared [header files](include/) for linking with the core application.
