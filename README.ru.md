# WMM-toolkit

| [English](README.md) | [Русский](README.ru.md) |
| -------------------- | ----------------------- |

Этот репозиторий служит **шаблоном и справочником** для разработки дополнений к [Wirus Mod Manager (WMM)](https://github.com/Procionn/WMM).
Вы можете клонировать его и использовать как основу для своего проекта.

Здесь вы найдете:
*   [Документацию](doc/ru/main.md) по разработке дополнений.
*   Подготовленные [заголовочные файлы (headers)](include/) для линковки с основной программой.

## Лицензия
Проект распространяется под лицензией MIT. Вы можете свободно копировать, распространять и изменять файлы этого репозитория. Единственное требование — сохранение уведомления об авторстве в производных работах.
